# Rikaraoke

<img src="../pikaraoke/logo.png" width="300" alt="Rikaraoke logo" />

A fork of [PiKaraoke](https://github.com/vicwomg/pikaraoke) by vicwomg, customized for Korean karaoke.

## What's different

- **Korean search support** — searches containing Korean (Hangul) characters automatically append `노래방` instead of `karaoke`, and bypass unidecode transliteration so results stay accurate
- Kept in sync with upstream pikaraoke releases

## Full documentation

See the [upstream README](https://github.com/vicwomg/pikaraoke) for installation instructions, usage, Docker setup, and troubleshooting.
